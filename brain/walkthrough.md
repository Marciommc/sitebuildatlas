# Walkthrough - Projeto Site Exithus (Entrega Final)

## Visão Geral
Este documento consolida a entrega do novo ecossistema digital da **Exithus Consultoria**. O projeto evoluiu de uma landing page para uma plataforma completa com área administrativa, gestão de leads e área de assinantes.

## Entregas Realizadas

### Fase 1: Site Institucional Moderno
*   **Tecnologia:** HTML5, CSS3 e JS (Puro), focado em performance máxima.
*   **Design:** Estética "Premium" (Dark Navy & Gold).
*   **Conteúdo:** Seções Hero, Sobre, Planos, Depoimentos e Contato (Integrado ao Piperun).

### Fase 2: Experiência e Conversão
*   **Motion UI:** Animações de entrada (Fade/Slide) ao rolar a página.
*   **Simulador:** Calculadora de potencial patrimonial em tempo real.
*   **WhatsApp:** Botão flutuante para contato rápido.
*   **SEO:** Dados estruturados (Schema.org) para buscas locais em Manaus.

### Fase 3: Sistema Integrado (Admin & Assinantes)
*   **Arquitetura:** Preparado para **Supabase (PostgreSQL)**.
*   **Modo de Teste:** Implementado sistema de persistência local (Mock DB) para validar fluxos completos sem necessidade de backend externo imediato.
*   **Funcionalidades:**
    *   **Login Unificado:** Acesso para Admin e Clientes (`login.html`).
    *   **Painel Admin:** Visualização de leads capturados no site e **Cadastro de Novos Assinantes** via modal.
    *   **Dashboard do Cliente:** Área restrita para visualização de documentos.

## Como Testar (Modo Demonstração)

1.  **Simulação de Lead:**
    *   Acesse a home (`index.html`), preencha o formulário de contato.
    *   O sistema salvará o lead no "Banco de Dados Interno".

2.  **Acesso Admin:**
    *   Vá para `login.html`.
    *   Entre com `admin@exithus.com` / `123`.
    *   Verifique se o lead criado aparece na tabela.
    *   Use o botão **"Novo Assinante"** para criar um acesso para um cliente.

3.  **Acesso Cliente:**
    *   Use o login criado no passo anterior (ou `cliente@email.com` / `123`).
    *   Acesse a área de documentos.

## Guia de Ativação (Produção)

Para conectar ao banco de dados real **Supabase**:

1.  **Crie sua conta:** Acesse [supabase.com](https://supabase.com) e crie um projeto.
2.  **Configure:**
    *   No arquivo `js/supabase-client.js`:
    *   Defina `MOCK_MODE = false`.
    *   Insira suas credenciais (`SUPABASE_URL` e `SUPABASE_KEY`).
3.  **Banco de Dados:** Rode o script SQL abaixo no painel do Supabase para criar as tabelas.

```sql
create table leads (
  id bigint generated by default as identity primary key,
  created_at timestamp with time zone default timezone('utc'::text, now()) not null,
  name text, email text, phone text, investment numeric
);

create table users (
  id bigint generated by default as identity primary key,
  email text unique, name text, role text default 'subscriber'
);

create table documents (
  id bigint generated by default as identity primary key,
  title text, url text, status text, user_email text
);
```
