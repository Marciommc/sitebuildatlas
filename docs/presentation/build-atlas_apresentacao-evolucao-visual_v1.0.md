# BUILD ATLAS — Evolução Visual e Entrega Fase 1 (v1.0)

## Objetivo
Consolidar uma identidade premium e institucional para o Build Atlas (Exithus Consultoria), com foco em:
- Autoridade patrimonial
- Captação qualificada de leads
- Base técnica pronta para evolução

## O que foi entregue (base técnica)
- Páginas legais: Privacidade, Cookies, Termos, Consentimento
- Banner LGPD/GDPR com persistência

- Sitemap atualizado

- Paleta oficial: Preto + Vermelho + Dourado (premium)

## O que falta para ficar “100% final”

- Imagens institucionais premium (hero e seções)
