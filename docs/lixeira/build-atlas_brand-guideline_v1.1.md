# BUILD ATLAS — Brand Guideline v1.1 (Preto + Vermelho + Dourado)

## Paleta Oficial
- Preto principal: #0A0A0A
- Preto secundário: #141414
- Vermelho institucional: #B11212
- Vermelho hover: #8E0F0F
- Dourado premium (accent): #C8A24C
- Branco: #FFFFFF
- Cinza texto: #B9B9B9
- Borda/linha: rgba(255,255,255,0.10)

## Tipografia
- Títulos: Playfair Display (Google Fonts)
- Texto: Inter (Google Fonts)

## Componentes
### Botão primário (CTA)
- BG: #B11212 | Texto: #FFF | Hover: #8E0F0F | Radius: 12px

### Botão secundário
- Borda: dourado 40% | Texto: dourado | Hover: fundo dourado 10%

### Chips/Badges
- BG: dourado 10% | Borda: dourado 25% | Texto: dourado

### Cards
- BG: preto secundário
- Borda: branca 10%
- Hover: borda dourada leve + brilho sutil

## Regras de uso do dourado
- Dourado é “acento premium”: usar em detalhes, não como cor dominante.
